# 🔐 Phishing Detector PRO

A web application that detects phishing URLs using security checks.

## 🚀 Features
- URL Risk Detection (High / Medium / Low)
- WHOIS Domain Analysis
- Analytics Dashboard (Charts)
- Clean UI

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Chart.js

## ▶️ How to Run

1. Clone repo
2. Install requirements:
pip install flask python-whois

3. Run:
python app.py

4. Open:
http://127.0.0.1:5000
